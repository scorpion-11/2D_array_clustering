# 2D_array_clustering
OBJECTIVE:
Displays the best cluster of students having maximum total score than specified average marks.

INPUT:
2D-Array having fields as studentName, studentRollNo, Test1 marks, Test2 marks, Test3 marks.

OUTPUT:
